# Sterile Compounding Supply Tracker

A batch scheduling, inventory, and purchase order tool for sterile compounding pharmacies.

## Features
- 📅 Monthly batch scheduling calendar
- ⚗️ Quick batch calculator
- 📦 Inventory tracking with shortfall alerts
- 🛒 Auto-generated purchase orders (copy or export CSV)

## Deploy to Vercel

### Option A — Vercel CLI (fastest)
```bash
npm install
npx vercel
```
Follow the prompts. Done.

### Option B — GitHub + Vercel Dashboard
1. Push this folder to a new GitHub repo
2. Go to vercel.com → "Add New Project"
3. Import the repo — Vercel auto-detects Vite
4. Click Deploy

## Local Development
```bash
npm install
npm run dev
```
